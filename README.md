## Try it yourself
+ `node --version === "v10.14.2"`
+ `npm install`
+ `npm start`
+ Open [http://localhost:8080](http://localhost:8080) in your browser

## Description
+ Online image labeling tool for machine learning research
+ Based on Node.js
+ Image data and labels are stored in data.csv
